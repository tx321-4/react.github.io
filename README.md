# 使用 create-react-app 快速构建 React 开发环境
create-react-app 是来自于 Facebook，通过该命令我们无需配置就能快速构建 React 开发环境。<br />
create-react-app 自动创建的项目是基于 Webpack + ES6 。<br />
执行以下命令创建项目：<br />
$ cnpm install -g create-react-app<br />
$ create-react-app my-app<br />
$ cd my-app/<br />
$ npm start<br />
